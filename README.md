# PONG
**Text-based pong recreation using C++/curses**

<u>Stable release: v1.1</u>
#
- Use `A/D` or arrows to slide ponger
- `q` to exit game
- `S` to start gameplay
#
Debug instance
- `p` for frame-by-frame gameplay
    - `n` or any input to advance frame'
- `o` for debug output
    - change verbosity at `getDBOut()`
